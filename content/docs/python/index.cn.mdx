1. Introduction
Description:
Learn how to install Python, write your first program, and use the Python interpreter.

2. Data Types
Description:
Work with numbers, strings, booleans, lists, tuples, dictionaries, and sets.

3. Control Flow
Description:
Use if, else, elif, loops, break, continue, and nested logic.

4. Functions
Description:
Create functions with arguments, return values, default and variable arguments.

5. File Handling
Description:
Read from and write to files using different modes and the with statement.

6. Object-Oriented Programming (OOP)
Description:
Learn classes, objects, inheritance, encapsulation, and special methods.

7. Error Handling
Description:
Handle errors using try, except, finally, and create custom exceptions.

8. Intermediate Python
Description:
Explore list comprehensions, lambdas, higher-order functions, and unpacking.

9. Advanced Python
Description:
Work with decorators, generators, iterators, context managers, and type hints.

10. Final Projects
Description:
Build small real-world projects using all the concepts learned in this course.
